java的历史以及地位

  java是一门面向对象的程序设计语言，1995年由sun公司发布，取名于叫java的coffee。2010年sun公司被Oracle公司收购。
  java的地位从两个方面可以看出。第一个方面：TIOBE的官方数据一定程度科学地展示某种编程语言的热度，2019年3月份java已经挤入第一名了，干掉了C和C++两位老大哥啊；第二个方面：网络已经成为信息时代最重要的交互媒介，那么基于网络的软件设计就成为软件设计领域的核心。java的平台无关性让java成为编写网络应用程序的佼佼者，而且java提供了许多以网络应用为核心的技术，使得java特别适用于网络应用软件的设计和开发。

java的平台无关性

​ java最大的优点就是平台无关性了吧，让它独领风骚。

  • JVM(Java Virtual Machine)：java虚拟机。是实现java平台无关性的关键。

  • java程序执行过程：Program.java源文件由编译器compiler生成Program.class字节码文件，再通过解释器interpreter(JVM)把字节码文件解释成对应平台上二进制机器指令。实现了一次编译，到处运行。

  • JDK(Java Develoment Kit),java语言的软件开发工具包。

    两个主要组件：

    javac–编译器，将源程序转成字节码,加后缀.java,文件名要与主类名一致

    java—运行编译后的java程序。注意不加后缀.class

  • JRE(Java Runtime Environment):包括java虚拟机(JVM)、java核心类库和支持文件。

如果只需要运行java程序，下载并安装JRE即可；如果要开发java软件，需要下载JDK。JDK中附带JRE。

JRE=JVM+javaSE标准类库

JDK=JRE+开发工具集(例如javac编译工具等)

java应用程序的基本结构

  一个java应用程序（也称为一个工程）是由若干个类构成的，这些类可以在一个源文件中，也可以分布在若干个源文件中。一个源文件由主类（含有main方法的类）和其他类构成。一个源文件编译后，它的每个类都会对应生成字节码文件.class，在同一目录下。

代码风格

  • 适当的注释

  • 类名，方法名，变量名取英文名并见名知其意。符合驼峰法命名规则(e.g. stuNameFunc)和Pascal命令规则(e.g. StuNameFunc)。

  • ALLmans风格：左右大括号各自独占一行（个人比较喜欢这种，但是好像很多官方代码都是Kernighan风格）

  • Kernighan风格：左大括号在上一行末尾，右大括号独占一行。
